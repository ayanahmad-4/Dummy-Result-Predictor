📊 GradeIQ — University Result Predictor & CGPA Planner

GradeIQ is a modern web-based tool designed to help students predict their semester results, calculate SGPI/CGPA, and plan academic goals with high accuracy (~90%).

Built using pure HTML, CSS, and JavaScript, it runs entirely in the browser — no installation required.

🚀 Features
🎯 Result Predictor
Enter:
Subject name
Credits (default: 4, editable)
CA marks (out of 60)
ESE marks (out of 40)
Automatically calculates:
Total percentage (CA + ESE)
Grade & grade points
Pass/Fail status
SGPI (Semester Grade Point Index)
📈 CGPA Goal Planner
Input:
Current CGPA
Completed semesters
Target CGPA
Remaining semesters
Get:
Required SGPI per semester
Estimated marks needed
Difficulty level (Easy / Moderate / Hard)
📚 Semester History Tracker
Store past semester SGPIs
Calculate overall CGPA
Track trends:
📉 Falling
📈 Rising
➖ Stable
🧮 Calculation Logic
🔹 Marks Calculation
Total % = CA (out of 60) + ESE (out of 40)
🔹 SGPI Formula
SGPI = Σ (Credit × Grade Points) ÷ Σ Credits
Failed or absent subjects = 0 grade points
